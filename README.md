# api-rick-and-morty

## Create Client
```
npm install -g @vue/cli 
```
## Create Project
```
create api-rick-and-morty
```
## Install Axios
```
npm install --save axios vue-axios 
```
## Install Bootstrap
```
npm install bootstrap@4.0.0-beta popper.js jquery --save-dev
```
### Compiles and hot-reloads for development
```
npm run serve
```
